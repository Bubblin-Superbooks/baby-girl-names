How to use:

Shortlist the first alphabets that you like. Then chew on those parts of the book to find a name you like:

  1. This book has over 1000 pages.
  2. All pages of the book are numbered
  3. Names are sorted alphabetically.
  4. Each name carries a search for meaning link.
  5. Each link also shows the frequency of a name i.e. number of girls with that name in last 100 years.
  6. Doubleclick or longtap to copy to clipboard.
  7. Once the book has been opened, it will continue to work offline without a live Internet connection.
  8. The book includes immigrant names as well.
  9. The manuscript of this book is available on Github under a suitable FOSS license.
  10. Girl names only. No boys!

Tabulated using the American Social Security records as of March 3, 2019.
