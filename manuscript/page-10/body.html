How to use:

Shortlist the alphabets that you like. Then chew through the book to find a name with those alphabets:

  1. The names are sorted alphabetically.
  2. All pages of the book are numbered
  3. This book has over a 1000 pages and it is growing.
  4. Once the book is turned to a specific page, it will remember the page you were on and continue to work offline without a live Internet connection.
  5. Each name carries a link to search for meaning of the name using DuckDuckGo.
  6. Each link also tooltips the frequency of a name i.e. number of girls with that name in the last 100 years.
  7. Doubleclick or longtap on the name to copy it to clipboard.
  8. The book includes immigrant names as well.
  9. The manuscript of this book is available on Github under a suitable FOSS license.
  10. Girl names only. No boys!

Reference: The American Social Security Records (March 3, 2019).
